# FYP_FinalThesis_ForXJTLUSAT301
在Feimax给的模板上变更了一些设计，使latex模板更贴近SAT301的word template.

Feimax原latex模板：https://github.com/feimax/latex_template_for_xjtlu_eee

# 主要更改内容

1. 封面格式
   - 调整主副标题大小
   - 加入上下小标题
   - 调整提交的学生信息和指导者等表格格式

2. 增加了章节
   - List of Figures
   - List of Tables
   - List of Acronyms
   - 并写入目录

3. 增加了Appendix章节

4. 分档分类
   - 将正文部分拆分为每一个章节为独立的 `.tex` 文件，提高了编写效率。
